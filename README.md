# 🎵 MoodyPlayer Backend

This is the **backend service** for MoodyPlayer – an AI-powered mood detection music player.  
It provides APIs to fetch mood-based songs using MongoDB + Express.js.

## 🚀 Tech Stack
- Node.js
- Express.js
- MongoDB + Mongoose
- ImageKit (for song images)
